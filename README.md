Passo a Passo:
1. Clone o repositório:

2. Acesse a pasta do projeto:
cd GS02-Web-Front
3. Instale as dependências:
npm install
npm install react-router-dom
npm install json-server
npm install react-icons
4. Execute o projeto
npm run dev
